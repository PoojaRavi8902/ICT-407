11.1 Manage Files Effectively

File management is the process of organizing, storing, naming, and retrieving files effectively. A good system ensures easy access, sharing, and prevents data loss.

  • Locate stored files: Use search functions or directory navigation. Example: finding a Word document by searching its name.
  • Open and import files: Applications can import multiple file types (.csv into Excel, .jpg into PowerPoint).
  • Save files in hierarchical folders: Example: School Work → Computer Science → Chapter 11 → Notes.docx.
  • Use meaningful filenames: Instead of “doc1.docx,” use “IGCSE_FileManagement_Notes.docx.”
  • Save/print in different formats: Document (.docx, .pdf), database reports, graphs/charts, web pages.
  • Export files in specific formats: .docx, .xlsx, .sdb, .pptx, etc.
  • Export in generic formats: .csv, .txt, .rtf, .pdf, .jpg, .png for compatibility across systems.

File Formats

Different file formats have unique characteristics and uses. Below are common ones:

.css – Stylesheets used for web design.
.csv – Comma-separated values, useful for spreadsheets/databases.
.gif – Animated/static images, small file sizes.
.htm/.html – Webpage content files.
.jpg – Compressed images, common for photos.
.pdf – Portable document format, fixed layout.
.png – High-quality images with transparency support.
.rtf – Rich text format, cross-platform text documents.
.txt – Plain text files.
.zip/.rar – Compressed file archives.

Need for generic file formats: Ensures files can be accessed across different platforms and software (e.g., exporting .csv so data can open in Excel, Google Sheets, or a database).

11.2 Reduce File Sizes for Storage or Transmission

Large files are often compressed to save storage space or make transmission quicker over the internet.

  • Compression tools: WinRAR, 7-Zip, WinZip.
  • Compressed formats: .zip and .rar are most common.
  • Advantages:
    • Saves storage space.
    • Speeds up file transfers.
    • Groups multiple files into one archive.
