1.1 Hardware and Software

Hardware

Hardware refers to the physical components of a computer system. These are tangible parts you can touch and see. Hardware can be classified into:

  • Internal Components: CPU (Central Processing Unit), processor, motherboard.
  • Internal Memory: RAM (Random Access Memory) – volatile memory used for temporary data storage while processing, ROM (Read-Only Memory) – non-volatile memory storing permanent instructions.
  • Expansion Cards: Graphics card, Sound card, Network Interface Card (NIC).
  • Input/Output Devices: Devices for entering data (keyboard, mouse, scanner) and receiving output (monitor, printer, speakers).
  • Storage Devices: Internal storage (HDD, SSD), external storage (USB drives, external HDDs).

Software

Software is a set of programs that control the operations of a computer and allow users to perform specific tasks.

  • Applications Software: Helps users complete tasks. Examples include:
    • Word processing (Microsoft Word)
    • Spreadsheets (Microsoft Excel)
    • Databases (Microsoft Access)
    • Control & measurement software
    • Graphics, video, and audio editing software
    • CAD (Computer Aided Design) applications
    • Mobile apps and applets
  • System Software: Allows the computer hardware to function and provides a platform for applications. Examples include:
    • Operating Systems (Windows, macOS, Linux)
    • Device drivers
    • Compilers and linkers
    • Utilities (antivirus, file managers)

Analogue vs Digital Data

Analogue Data: Continuous signals representing physical quantities, e.g., sound waves, temperature readings.

Digital Data: Discrete signals using binary code (0s and 1s) to represent information.

Computers require:

  • Conversion of analogue → digital for processing
  • Conversion of digital → analogue to control devices (e.g., speakers, motors)

Example Table – Analogue vs Digital

Feature Analogue Digital
Signal Continuous Discrete (Binary)
Example Sound waves MP3 files, computer data
Accuracy Less precise, can degrade Highly accurate, consistent
Storage Hard to store for long periods Easy to store digitally

1.2 The Main Components of Computer Systems

Central Processing Unit (CPU)

The CPU is the brain of the computer. It executes instructions, processes data, and coordinates all other components.

  • Control Unit (CU): Directs the flow of data and instructions inside the CPU.
  • Arithmetic Logic Unit (ALU): Performs all calculations and logical operations.
  • Registers: Small, high-speed storage locations within the CPU used for temporary data storage during processing.

Internal Memory

Memory inside the computer is used to store data and instructions temporarily or permanently.

  • RAM (Random Access Memory): Volatile memory used for storing data and programs currently in use. Cleared when the computer is turned off.
  • ROM (Read-Only Memory): Non-volatile memory storing permanent instructions like firmware. Cannot be easily modified.

Differences Between ROM and RAM

Feature RAM ROM
Volatility Volatile – loses data when power is off Non-volatile – retains data permanently
Purpose Stores data and programs in use Stores firmware and permanent instructions
Read/Write Can read and write Usually read-only
Capacity Typically larger (GBs) Smaller (MBs)

Input and Output Devices

These devices allow interaction between the user and the computer.

  • Input devices: Keyboard, mouse, scanner, microphone, webcam.
  • Output devices: Monitor, printer, speakers, projectors.

Difference: Input devices send data to the computer, output devices receive data from the computer.

Backing Storage

Backing storage provides permanent storage for data and programs.

  • Hard Disk Drives (HDD)
  • Solid State Drives (SSD)
  • Optical disks (CD, DVD)
  • Flash drives/USB drives

Difference from Internal Memory: Backing storage retains data when the computer is off, whereas RAM loses data when power is removed.

Example Table – Storage Comparison

Feature Internal Memory (RAM/ROM) Backing Storage
Volatility RAM: Volatile, ROM: Non-volatile Non-volatile
Speed Very fast Slower than RAM
Capacity Smaller (MBs–GBs) Larger (GBs–TBs)
Purpose Temporary or permanent instructions for processing Permanent storage of files and programs

1.3 Operating Systems

What is an Operating System (OS)?

An Operating System (OS) is a type of system software that manages computer hardware and software resources, and provides services for application software. It acts as an intermediary between the user and the computer hardware.

Characteristics of Operating Systems

  • Command Line Interface (CLI): User interacts by typing commands. Example: MS-DOS, Linux terminal.
  • Graphical User Interface (GUI): User interacts with graphical icons, menus, and windows. Example: Windows, macOS.
  • Dialogue-based Interface: Communicates with users through prompts and dialogue boxes. Example: installation wizards.
  • Gesture-based Interface: Allows users to control devices using gestures or touch. Example: smartphones, tablets, interactive screens.

Functions of an Operating System

  • Managing hardware resources (CPU, memory, storage, input/output devices)
  • File management (creating, reading, updating, and deleting files)
  • Task management (multitasking, scheduling processes)
  • Providing a user interface (CLI, GUI, etc.)
  • Security and access control

Differences Between Types of Operating Systems

Type of OS Advantages Disadvantages Example
CLI Faster for experienced users, low system resources needed Difficult to learn for beginners, no visual aids MS-DOS, Linux Terminal
GUI User-friendly, intuitive, visually rich Consumes more system resources, slower for repetitive tasks Windows, macOS
Dialogue-based Guided, easy for installation or setup tasks Limited functionality, not suitable for complex operations Installation wizards
Gesture-based Intuitive for touch-enabled devices, supports natural interaction Requires specialized hardware, not precise for complex tasks Smartphones, Tablets

Key Points to Remember

  • An OS controls all hardware and software on a computer.
  • Choice of OS affects user experience, system speed, and usability.
  • Some OS types are better suited for certain devices (e.g., GUI for desktops, gesture-based for mobile devices).

1.4 Types of Computers

Desktop Computers

Desktop computers are stationary computers designed for regular use at a single location.

  • Characteristics: Non-portable, larger screen and keyboard, easily expandable with additional hardware (e.g., RAM, graphics card).
  • Uses: Office work, business management, education, gaming, multimedia, entertainment.
  • Advantages: High processing power, large storage, easily upgraded.
  • Disadvantages: Not portable, requires a constant power supply.

Mobile Computers

Mobile computers are portable computing devices that can operate on battery power and can be used in various locations.

  • Examples: Laptops, smartphones, tablets, phablets.
  • Characteristics: Compact, lightweight, portable, battery-powered, often touchscreen-enabled.
  • Uses: Office work, education, gaming, entertainment, remote control of devices, mobile applications.
  • Advantages: Portable, convenient for on-the-go use, can access cloud or network resources anywhere.
  • Disadvantages: Limited processing power compared to desktops, harder to upgrade, smaller storage.

Comparison Table – Desktop vs Mobile Computers

Feature Desktop Computer Mobile Computer
Portability Non-portable Portable
Processing Power High Moderate
Upgradeability Easy to upgrade hardware Limited or difficult to upgrade
Battery Dependency Requires constant power supply Can run on battery
Use Cases Office, gaming, multimedia, education On-the-go work, remote learning, mobile apps, entertainment

Key Points to Remember

  • Desktops are suitable for stationary workstations requiring high power and expandability.
  • Mobile computers provide flexibility, portability, and convenience for users who move frequently.
  • Choice between desktop and mobile depends on user needs, environment, and performance requirements.

1.5 Emerging Technologies

Introduction

Emerging technologies are new and innovative technologies that are changing how we live, work, and interact with the world. They are rapidly evolving and often integrate with existing systems to improve efficiency and capabilities.

Artificial Intelligence (AI)

AI enables machines to simulate human intelligence, including learning, reasoning, problem-solving, and decision-making.

  • Examples: Chatbots, virtual assistants (Siri, Alexa), self-driving cars, predictive analytics.
  • Impact on everyday life: Automates repetitive tasks, enhances decision-making, provides personalized services, improves safety and efficiency.

Extended Reality (XR)

Extended Reality includes Virtual Reality (VR) and Augmented Reality (AR).

  • Virtual Reality (VR): Fully immersive digital environment where users interact with a simulated world. Example: VR gaming, training simulators.
  • Augmented Reality (AR): Overlays digital information onto the real world. Example: AR apps for education, navigation, or interior design.
  • Impact on everyday life: Enhances learning, training, gaming, and entertainment experiences. Provides interactive and engaging interfaces for users.

Other Emerging Technologies

  • Internet of Things (IoT) – connecting devices to the internet for remote control and monitoring (smart homes, wearable devices).
  • Robotics – automated machines for manufacturing, healthcare, and services.
  • 3D Printing – rapid production of objects in various industries, including medical and construction.

Key Points to Remember

  • Emerging technologies are continuously evolving and transforming our daily lives.
  • They improve productivity, accessibility, and user experience across various fields.
  • Students should understand both benefits and potential challenges, including ethical and social implications.
