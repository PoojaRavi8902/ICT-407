Part B — Short Answer Questions (15)

1. Define file management.
Model Answer: The process of organising, storing, retrieving and maintaining files on a computer system.
2. Give one reason why file extensions are important.
Model Answer: They indicate the file type and which program can open it.
3. State one example of a compressed file format.
Model Answer: .zip, .rar, or .7z
4. What does the term 'directory' mean?
Model Answer: Another term for a folder, used to organise files.
5. Why is backing up data important?
Model Answer: To prevent data loss due to hardware failure, accidental deletion, or malware.
6. Name one file system used in Windows.
Model Answer: NTFS or FAT32
7. What is meant by the term 'read-only file'?
Model Answer: A file that can be viewed but not modified.
8. State one advantage of cloud storage.
Model Answer: Files can be accessed from anywhere with an internet connection.
9. What is meant by 'file fragmentation'?
Model Answer: When parts of a file are stored in non-contiguous clusters, slowing access speed.
10. Define 'defragmentation'.
Model Answer: The process of reorganising fragmented data to improve file access speed.
11. Give an example of an optical storage device.
Model Answer: CD, DVD, Blu-ray
12. State one reason why a file may become corrupted.
Model Answer: Power failure during save, virus attack, or hardware failure.
13. What is metadata in relation to a file?
Model Answer: Data about data, e.g. file size, creation date, author.
14. Define the term 'backup'.
Model Answer: A copy of data kept separately to allow recovery in case of data loss.
15. Name one removable storage device.
Model Answer: USB flash drive, external hard drive, or memory card.
