Teaching Notes

21.1 Web Page Structure

Purpose of HTML: HTML (HyperText Markup Language) defines the structure and meaning of your webpage content. It tells the browser what each piece of content represents.

Elements & Tags: Elements are created using tags. Most tags have an opening and closing tag. Example: <p>This is a paragraph</p>.

Nesting: Tags can be placed inside other tags, but must close in opposite order (LIFO).

Basic HTML Document Structure:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
    <meta charset="UTF-8">
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is my first webpage.</p>
  </body>
</html>

Key Concepts to Teach:

  • Block vs inline elements
  • Whitespace and indentation conventions
  • Importance of <head> metadata
21.2 Text Formatting

HTML provides structural and stylistic text formatting.

  • Heading Tags: <h1> to <h6>, where <h1> is largest.
  • Paragraph: <p> creates blocks of text.
  • Bold: <b> (visual), <strong> (semantic).
  • Italic: <i> (visual), <em> (semantic emphasis).
  • Line Break: <br> forces a new line.

Example:

<p>This is <strong>important</strong> text.</p>
21.3 Hyperlinks

Hyperlinks connect one webpage to another.

Absolute links: Point to external sites.
<a href="https://www.cie.org.uk" target="_blank">Visit Cambridge</a>

Relative links: Used inside your website folder.
<a href="about.html">About Us</a>

Anchors: Jump to a section on the same page.

<a href="#section1">Go to Section 1</a>
...
<h2 id="section1">Section 1</h2>
21.4 Images

Images improve the visual appeal of webpages.

  • src: Path to image file.
  • alt: Text shown if image fails; used by screen readers.
  • width / height: Resizes the image.
<img src="images/logo.png" alt="School Logo" width="200">

Discuss JPG vs PNG vs GIF.

21.5 Tables

Tables display data in rows and columns.

<table border="1" cellpadding="5">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Ali</td>
    <td>16</td>
  </tr>
</table>

Teach table headers, spanning, and layout best practices.

21.6 Forms

Forms capture user input.

  • Text fields – <input type="text">
  • Radio buttons
  • Checkboxes
  • Dropdown lists
  • Textarea
  • Submit button
<form action="send.php" method="post">
  Name: <input type="text" name="username"><br>
  Gender:
  <input type="radio" name="g" value="M">Male
  <input type="radio" name="g" value="F">Female<br>
  <input type="submit" value="Send">
</form>
21.7 CSS Basics

CSS controls the appearance of webpages.

Inline CSS: In the element.
<p style="color:red;">

Internal CSS: In <style> inside <head>.

External CSS: Separate .css file linked in HTML.

p {
  font-size: 18px;
  color: blue;
}
21.8 Website Folder Structure

Teach organising files:

  • /images
  • /css
  • /scripts
  • /pages

Discuss naming conventions, linking resources, and avoiding broken links.
