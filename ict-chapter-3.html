3.1 Storage Devices vs Storage Media

Storage device ‚Üí The physical hardware used to read from and write to storage media.
Example: CD/DVD drive, hard disk drive (HDD), SSD.

Storage media ‚Üí The material or medium where data is stored.
Example: CD, DVD, Blu-ray disc, magnetic tape, flash memory card.

üëâ A storage device often requires storage media to function. For example, a DVD drive (device) reads and writes to a DVD disc (media).
üëâ Some devices are self-contained with built-in media, like an SSD or HDD.

3.2 Categories of Storage Technology

There are three main storage technologies:

A. Magnetic Storage

Examples: Hard disk drives (fixed & portable), magnetic tape.

How it works: Data is stored by magnetising tiny spots on a metal or glass disk surface. A read/write head hovers above the spinning disk to read or change these spots.

Magnetic Hard Disk Drives (HDDs)

  • Fixed HDDs: Built inside desktop/laptop
  • Portable HDDs: External drives, connected via USB/Thunderbolt

Characteristics:

  • Large storage capacity (terabytes common).
  • Cheaper per gigabyte compared to SSD.
  • Slower than SSDs (mechanical moving parts).
  • Prone to damage if dropped (fragile).

Uses:

  • Storing operating systems, applications, personal files.
  • Backups (external HDDs).
  • Cloud servers (bulk data).

Advantages:

  • Very high capacity.
  • Cost-effective for large storage.
  • Can be used many times (re-writable).

Disadvantages:

  • Slower access speed compared to SSD.
  • Mechanical parts ‚Üí wear & tear, risk of damage.
  • Heavier and less portable than SSDs.

Magnetic Tape

Stored on reels or cartridges.
Sequential access (must go through tape in order).

Characteristics:

  • Very high capacity (used in terabytes).
  • Very slow access time (not random access).
  • Long-lasting, reliable if stored correctly.

Uses:

  • Backup and archiving (banks, governments, large companies).
  • Not for everyday use by individuals.

Advantages:

  • Cheapest per gigabyte (for bulk storage).
  • Durable and long-lasting.
  • Good for archiving rarely accessed data.

Disadvantages:

  • Very slow to access.
  • Requires tape drive (expensive).
  • Not practical for small-scale storage.
B. Optical Storage

Examples: CD, DVD, Blu-ray.

How it works: Uses lasers to read/write pits and lands on the surface of a disc.

CD (Compact Disc)

  • Capacity: ~700 MB.
  • Types: CD-ROM (read-only), CD-R (recordable once), CD-RW (rewritable).

Uses: Music, software distribution, small data storage.

DVD (Digital Versatile Disc)

  • Capacity: 4.7 GB (single-layer), 8.5 GB (dual-layer).
  • Types: DVD-ROM, DVD-R, DVD-RW.

Uses: Movies, games, software, data backup.

Blu-ray Disc

  • Capacity: 25 GB (single-layer), 50 GB (dual-layer).
  • Uses a blue laser (shorter wavelength ‚Üí more data storage).

Uses: High-definition movies, large games, data storage.

Characteristics (all optical):

  • Portable, lightweight.
  • Susceptible to scratches/damage.
  • Slower access compared to HDD/SSD.

Advantages:

  • Cheap, portable.
  • Easy to distribute media (music, films, software).
  • Some are rewritable.

Disadvantages:

  • Lower capacity compared to modern HDD/SSD.
  • Requires optical drive.
  • Becoming obsolete (USB & online distribution replacing them).
C. Solid-State Storage

Examples: SSD, pen drives, flash drives, memory cards.

How it works: Uses flash memory chips (no moving parts). Data stored as electrical charges in transistors.

Solid-State Drives (SSD)

  • Fixed SSD: Built into laptops and desktops.
  • Portable SSD: External drives connected via USB.

Characteristics:

  • Very fast access speeds (boots OS in seconds).
  • More expensive per GB than HDDs.
  • Lightweight, durable (no moving parts).

Uses:

  • Operating systems.
  • Gaming laptops and ultrabooks.
  • Portable high-speed storage.

Advantages:

  • Very fast read/write.
  • Silent, shock-resistant.
  • Lightweight and compact.

Disadvantages:

  • More expensive.
  • Lower maximum storage compared to HDDs.

Flash Drives / Pen Drives

  • Portable USB sticks.
  • Capacity: 2 GB ‚Äì 1 TB.

Uses: File transfer, backup, carrying work.

Memory Cards

  • Types: SD card, microSD, xD, CFast.
  • Capacity: Ranges from MB to hundreds of GB.

Uses: Digital cameras, smartphones, tablets, drones, games consoles.

Advantages (all solid-state media):

  • Highly portable.
  • Fast access.
  • Durable.

Disadvantages:

  • Higher cost per GB.
  • Smaller (easier to lose).
  • Limited read/write cycles (eventually wear out).
