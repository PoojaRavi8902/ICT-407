12.2 Editing Images

Editing allows you to adjust an image so it fits the context of your document or presentation. Common edits include resizing, cropping, rotating, wrapping text, and adding borders.

12.2.1 Resize an Image

To maintain proportions, always keep aspect ratio locked. Example:

Resize snowball.jpg to 8cm high while maintaining aspect ratio.

12.2.2 Wrap Text Around an Image

  • In Line with Text: acts like a character
  • Square: text flows around the image
  • Tight: text hugs the image closely
  • Through: flows inside transparent areas
  • Top & Bottom: only above and below
  • Behind Text: background watermark style
  • In Front of Text: overlaps the text

12.2.6 Crop an Image

Cropping means removing part of an image. Unlike resizing, it changes the aspect ratio without distorting the picture. This is useful to focus on the important area or remove unwanted background.

  1. Right click on trees.jpg and select Crop.
  2. Drag the crop handles (black bars at the corners/edges) so that about 25% of the image is cut away.
  3. Click outside the image to apply the crop. The snow will now appear at the top of the branches.

12.2.8 Adjust Colour Depth

Colour depth = number of bits used to represent colour per pixel. Higher depth = more detail, larger file size.

  • 8-bit per channel ‚Üí 24-bit colour (JPEG standard).
  • 16-bit per channel ‚Üí 48-bit colour (PNG supports this).

In Photoshop: Image ‚Üí Mode ‚Üí 16 Bits/Channel. Save as trees2.png. Compare file size with trees.png.

‚ö†Ô∏è Use higher colour depth before heavy editing to reduce blurring or artefacts.
